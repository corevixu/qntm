# Introduction
This webapp utilizes nextjs framework with custom made react components.

# Blog/News Fecthing
Using google search engine and automated nextjs workers, news is fetched regularly and updates automatically.

# Stock Analysis
We have used yahoofinance and tradingview with some custom made react components to give calculated stock analysis of selected quantum computing stocks.

# Ai chat
It also has integrated page using external models for text-based, image-based and voice-based communications and guidance.
Custom made react components for different types of conversations.
